MPI编程(一)-mpich安装

发表于2019-11-12|更新于2021-02-15|计算机学习
|字数总计:100|阅读时长:1分钟|阅读量:

在ubuntu18.04安装mpich

1.下载

mpich官网

2.解压

tar -zxvf mpich-3.3.1.tar.gz

3.检查安装环境

./configure -prefix=/home/mpi/mpich2

配置安装位置为 /home/mpi/mpich2

4.编译

make

5.安装

sudo make install

6.添加环境变量

vi ~/.bashrc

如果有就不要添加了(注意修改配置文件目录)

7.测试

which mpicc
